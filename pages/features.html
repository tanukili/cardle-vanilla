# 原子化的筆記方式

Cardle 是一個基於卡片筆記（Zettelkasten）開發的筆記空具，協助你快速建立卡片與卡片見的連結。同時包含學習資源整合與進度追蹤功能，讓你的學習更迅速直覺。

打造你的專屬知識庫

# 輕鬆整理知識

從今天開始，用卡片累積知識、整理思路。把想法寫成卡片，把知識連成脈絡。你可以透過標籤將卡片依做個人化的別類，將分散的學習內容將化為有結構的知識庫。

Cardle 幫你把碎片化資訊，變成能長期成長的個人知識庫。

# 留住上次停下的腳步

集中整理你正在閱讀的文章、書籍、課程或影片，針對每項資源標記章節、頁碼或時間軸進度。系統會圖表化顯示學習進度，並自動提醒尚未完成的項目。

Cardle 讓學習進度一目了然、不再遺忘。
